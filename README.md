# 🖤 Model Portfolio — Angelina Love

A modern, elegant model portfolio website built with React + TypeScript.

## 🌐 Live Demo
Coming soon on Azure!

## 🛠️ Built With
- React 18
- TypeScript
- Vite
- React Router DOM
- React Icons
- CSS3 (custom styling)

## ✨ Features
- Full screen video background
- Black & white photo gallery with lightbox
- Masonry grid portfolio page
- Smooth page navigation with React Router
- Social media links (Instagram, TikTok, Facebook)
- Custom floating navbar

## 📁 Project Structure
src/
├── Header/
├── Body/
├── About/
├── Portfolio/
└── Footer/

## 🚀 Getting Started

npm run dev
